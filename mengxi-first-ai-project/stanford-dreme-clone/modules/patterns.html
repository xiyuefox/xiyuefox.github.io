🔄 模式与序列

帮助儿童识别和延续各种模式的教育资源

1. AB模式识别
核心概念：简单模式
教儿童识别和延续ABABAB等简单交替模式。
2. AAB模式练习
核心概念：重复模式
练习识别和延续AABAAB等重复模式。
3. 颜色模式创作
核心概念：模式创作
让儿童使用颜色卡片创作自己的模式。
4. 形状模式识别
核心概念：形状模式
识别和延续由不同形状组成的模式。
5. 增长模式学习
核心概念：增长模式
学习识别和延续如1, 2, 3, 4...等增长模式。
