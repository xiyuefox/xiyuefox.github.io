Diane: AB Pattern
儿童：Diane
核心概念：AB重复模式
Diane观看了一个黄-紫-黄-紫的交替模式，然后她重新创建并延续了这个模式。
Diane: Explain and Extend a Pattern
儿童：Diane
核心概念：模式解释与延续
Diane刚把黄色和紫色的立方体沿着桌子排列好。当被问及她是怎么知道该怎么做时，她说：“Make a pattern.”。当被要求描述时，她说是“yellow-purple-yellow-purple”。然后她使用重复单元来说明下一个立方体是什么颜色。
Meredith: Explain a Pattern
儿童：Meredith
核心概念：模式解释
Meredith解释说，她正在看的AB模式是一个模式，因为“it has the same thing”。当被要求进一步描述时，她说同样的东西是“yellow-blue-yellow-blue-yellow-blue”。
Sonia: Identify Unit of Repeat
儿童：Sonia
核心概念：重复单元识别
Sonia观看了一个涉及虚构电影角色的重复模式并延续了它。当被问及她是怎么知道的时，她说：“Because, look”，然后用手将其分成三个重复单元。然后她确定每个组中第一个出现的角色，并说它们是相同的。
Brian: Describes a Growing Pattern
儿童：Brian
核心概念：增长模式
Brian刚制作了增长模式中的第四个塔。他解释说，他知道如何制作第四个塔，因为它以2的数量增长，并且立方体是蓝白交替的。
Ben: Missing Term
儿童：Ben
核心概念：缺失项识别
当一张卡片从重复模式中移除时，Ben被要求闭上眼睛。他正确地识别出这张卡片是一只青蛙，并解释了他是如何根据模式重复的方式知道这一点的。
Meredith: Equivalent Pattern (Colors)
儿童：Meredith
核心概念：等效模式（颜色）
Meredith观看了一个新的AAB模式，并被要求使用不同的颜色复制该模式。她做到了，定期地从她正在复制的模式和她正在创建的模式之间来回查看，以检查她的工作。最后，她将它们并排排列，以显示它们是等效模式。
Ben: Equivalent Pattern (Cards)
儿童：Ben
核心概念：等效模式（卡片）
Ben成功地使用不同的卡片复制了现有的AB卡片模式。
