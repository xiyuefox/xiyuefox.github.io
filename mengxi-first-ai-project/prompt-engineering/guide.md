# CodeSignal 风格 Prompt Engineering 提示词工程课程

## 课程概述
本课程系统教授 Prompt Engineering 的核心概念、技术和最佳实践，帮助学习者掌握如何与 AI 模型进行高效交互，生成高质量、符合预期的输出。课程结合理论讲解与大量实践案例，覆盖从基础到进阶的全部内容。

## 学习目标
1. 理解 Prompt Engineering 的定义和核心价值
2. 掌握设计高质量提示词的 7 大核心原则
3. 熟练运用 5 种关键 Prompt 技术（Zero-Shot、Few-Shot、CoT 等）
4. 学会避免常见的 Prompt 设计误区
5. 能够针对不同任务类型（代码、写作、分析等）设计专业 Prompt
6. 掌握迭代优化 Prompt 的方法

## 核心模块

### 模块 1：基础概念与原则
#### 1.1 什么是 Prompt Engineering？
- 定义：设计、优化和测试与 AI 模型交互的输入文本
- 价值：连接人类意图与 AI 能力的桥梁

#### 1.2 7 大核心设计原则
##### 1.2.1 清晰具体（Clarity & Specificity）
- 避免模糊性：明确任务目标、格式要求和约束条件
- **示例对比**：
  - 差：「写一首诗」
  - 好：「写一首 4 行日式俳句，主题为春天的樱花，突出其转瞬即逝的美感」

##### 1.2.2 提供上下文与背景（Context & Background）
- 为模型补充必要的前置信息（领域知识、场景设定）
- **示例**：
  - 提示词：「作为资深前端工程师，请用 3 个生活化类比解释 React 中的 Hooks，假设听众是完全不懂编程的产品经理」


### 3. 明确指令与格式（Explicit Instructions）
- 直接告诉模型**要做什么**和**输出格式**（如列表、段落、JSON、代码）。
- **示例**：
  - 差：「分析这个问题」
  - 好：「分析以下用户投诉的核心诉求，并以 bullet point 列出改进建议：'我的订单 3 天还没发货，客服电话一直打不通'」


## 三、关键技术
### 1. 零样本学习（Zero-Shot Learning）
- 无需示例，直接让模型完成任务（依赖模型的预训练知识）。
- **示例**：「将以下句子翻译成法语：'人工智能正在改变世界'」


### 2. 少样本学习（Few-Shot Learning）
- 提供少量示例（1-5 个），让模型学习任务模式后再执行。
- **示例**：
  ```
  示例 1：
  中文：猫喜欢吃鱼
  英语：Cats like to eat fish

  示例 2：
  中文：狗喜欢啃骨头
  英语：Dogs like to chew bones

  请翻译：中文：鸟喜欢唱歌
  ```


### 3. 思维链（Chain-of-Thought, CoT）
- 要求模型**展示推理步骤**，而非直接给出答案，提升复杂任务（如数学、逻辑）的准确性。
- **示例**：
  - 提示词：「书架上有 100 本书，周一卖了 30 本，周二买了 50 本，周三卖了 25 本。现在书架上有多少本书？请展示计算步骤。」
  - 输出：「步骤 1：周一后剩余 100-30=70 本；步骤 2：周二后变为 70+50=120 本；步骤 3：周三后剩余 120-25=95 本。答案：95 本」


### 4. 自我一致性（Self-Consistency）
- 对同一问题生成多个推理路径，选择出现频率最高的结果，降低随机性。
- **适用场景**：数学题、逻辑推理等需要高准确性的任务。


## 四、最佳实践
1. **使用角色设定**：为模型赋予身份（如「作为专业律师」「作为小学老师」），约束输出风格。
2. **分段清晰**：用换行、分隔符（如 ---）区分任务、上下文和输入内容。
3. **迭代优化**：测试初始提示，根据输出调整指令（如补充细节、明确格式）。
4. **避免假设**：不要认为模型知道特定领域的最新信息或私有数据。
5. **限制输出长度**：对长文本任务，明确要求「不超过 500 字」或「分 3 点回答」。


## 五、常见误区
- ❌ 过于简洁：「帮我写报告」→ 模型不知道报告主题、长度、受众。
- ❌ 模糊术语：「优化这个代码」→ 模型不知道"优化"是指性能、可读性还是功能。
- ❌ 复合任务：一次让模型完成太多任务（如「写文案+设计海报+生成代码」）→ 降低各任务质量。


## 六、示例应用
### 1. 代码生成
> 提示词：「写一个 Python 函数，用快速排序算法对整数列表进行升序排序，包含注释和 3 个测试用例。」

### 2. 创意写作
> 提示词：「写一个 500 字的科幻故事，主题是机器人守护末日图书馆，突出它对人类情感的好奇。」

### 3. 数据分析
> 提示词：「分析以下销售数据（2024 年 1-3 月：10 万/15 万/22 万），用 3 句话总结增长趋势，并给出 2 个改进建议。」
