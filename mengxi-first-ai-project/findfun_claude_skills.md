
# Claude Skills 核心探索笔记
## 核心主题
系统梳理 Claude Skills 的定义、功能、使用方式与应用价值

## 关键发现与思考点
### 1. 本质定位
- Skills 是可动态加载的指令/脚本资源集合，用于提升 Claude 专项任务表现
- 核心解决：复杂场景下的任务标准化与知识过载问题
- 覆盖范围：跨平台支持所有 Claude 应用场景（聊天、API、Claude Code）

### 2. 核心特性
- 渐进式披露：仅加载任务相关内容，避免上下文冗余
- 自动激活：Claude 智能匹配并调用相关 Skills
- 脚本支持：可嵌入可执行脚本实现高级功能
- 全平台兼容：跨 Claude 生态系统可用

### 3. 使用与访问
- 适用计划：Pro/Max/Team/Enterprise（需启用代码执行）、Claude Code Beta、API 用户
- 启用条件：必须在设置中开启代码执行功能
- 调用方式：完全自动化，无需手动触发

### 4. 类型与应用场景
#### 内置 Skills（Anthropic 维护）
- 示例：Excel/Word/PowerPoint/PDF 增强文档创建
- 特点：开箱即用，覆盖通用高频场景

#### 自定义 Skills（用户/企业创建）
- 品牌化文档生成、公司专属工作流执行、JIRA/Asana 任务自动化、个人效率工具

### 5. 价值与优势
- 提升任务性能：针对专项任务的定制化指导
- 沉淀组织知识：固化公司流程与最佳实践
- 降低使用门槛：无需编码即可创建简单 Skills
- 扩展性强：支持脚本实现复杂需求

### 6. 限制与注意事项
- 处于 Beta/预览阶段，功能未完全 GA
- 依赖代码执行权限，需注意安全配置
- 需明确区分与 Projects/MCP/Custom Instructions 的使用场景
