# Obsidian Tool Usage Guide

## Available Tools:

1. **mcp__obsidian__list-available-vaults** - List all available vaults
2. **mcp__obsidian__search-vault** - Search for content in a vault
3. **mcp__obsidian__read-note** - Read content of a note
4. **mcp__obsidian__create-note** - Create a new note
5. **mcp__obsidian__edit-note** - Edit an existing note
6. **mcp__obsidian__create-directory** - Create a directory in vault
7. **mcp__obsidian__add-tags** - Add tags to notes
8. **mcp__obsidian__remove-tags** - Remove tags from notes
