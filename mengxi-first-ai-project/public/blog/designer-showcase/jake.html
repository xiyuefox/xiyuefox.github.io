INTERVIEW_ARCHIVE / THE_LUDDITE_VOICE

> ON_SLOWNESS (关于慢下来)

"我的设计理念是向那些习以为常的数据驱动型设计发起挑战，**鼓励人们慢下来**。我鼓励浏览者去互动，不要担心如果没能立即呈现信息就会失去他们。"


> ON_REPAIRABILITY (关于完全控制)

"我几乎总是从零开始构建代码。这很重要：我必须理解代码里发生的一切，我有能力在出问题时修复它。就像我的自行车，出了问题我能亲手修好。"


> ON_THE_OLD_WEB (关于旧互联网的纯真)

"我怀念 2010 年以前的互联网。它被社交网络囚禁并拆解了。我是在论坛上长大的。搜索引擎曾能索引一切，而现在信息在私人网络里丢失了。"


> ON_RESTRICTION (关于限制带来的成长)

"在工作中，我依靠限制而成长。帕金森定律说工作会自动膨胀占满时间，我深信不疑。我热爱那些能像玩具一样被成年人拿来玩的东西。"


> ON_LANDSCAPE (关于 ViaMichelin 效应)

"我用一个叫 ViaMichelin 的 App，因为它有『风景路线』选项。它绕过高速公路，带你经过村庄和山脉。它给了你思考的时间，让你偶遇驻足点。"

VISION_SPEC / THE_7_HABIT_SHIFTS

"浏览器的未来不是更快，而是更懂你。"

01_READING → MAPPING
从一次性阅读 → 自动画出脑图：哪个网页讲了什么、跟你以前读过的哪篇有关，自动串起来。
02_JUMPING → TRACING
从乱跳网页 → 记录思路分支：每一步操作都能记录，想回头看某条思路分支，随时回去。
03_KEYWORD → SEMANTICS
从傻搜关键词 → 理解搜索意图：能理解上下文，从页面中自动提取数据、图表与结论。
04_SOLO → MULTIPLAYER
从一个人刷网页 → 边看边聊的协作：让网页像 Docs 一样可以一起标注、评论、实时查阅。
05_PASTE → INTELLECTUAL_REF
从复制粘贴 → 随时落笔并自动引用：直接选中写感想，系统自动带上精准的原文出处。
06_FOLDER → AUTO_STRUCTURE
从手动建文件夹 → 内容自觉生长：根据阅读习惯与引用关系，自动长出知识结构树。
07_CLICK → SPATIAL_UNIVERSE
从点击网页 → 在信息宇宙里移动：像逛 Google Earth 那样操控空间，感受信息的物理体积。
