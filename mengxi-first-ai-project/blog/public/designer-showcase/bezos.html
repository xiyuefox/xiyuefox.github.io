AWS_CONSTITUTION / API_MANIFESTO

AWS 宪法：强制性的解耦

01 所有团队必须通过接口 (API) 公开其数据和功能。
02 团队必须通过这些 API 彼此沟通，绝不允许其他形式的进程间通信。
03 没有任何后门：不允许直接访问数据库，不允许共享内存。
04 技术栈不限：HTTP, Corba, Pubsub 均可，只要接口标准。
05 所有 API 必须从头设计为“可外部化”的，即必须能卖给外部客户。
06 任何不这样做的人都会被解雇。谢谢，祝你愉快。

这是将“内部工具”转化为“全球基础设施”的物理门槛。
